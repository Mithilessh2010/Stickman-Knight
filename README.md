# Stickman Duel Arena

A fast-paced 1v1 stickman arena fighter built with React + Vite + HTML5 Canvas.

## Run locally
```
npm install
npm run dev
```
Open http://localhost:5173

## Build
```
npm run build
```
Outputs to `dist/`.

## Deploy to Vercel
Push the folder to a Git repo and import in Vercel — it auto-detects Vite. No configuration needed.

## Controls
- A / D — move
- W — jump
- J — basic attack
- K — ability 1
- L — ability 2
- U — ultimate

## Characters
- Sword Fighter — Slash · Lunge · Riposte (parry) · Whirlwind
- Spear Fighter — Thrust · Pole Vault · Javelin · Spear Rain
- Mage — Arcane Bolt · Blink · Frost Nova · Meteor
- Brute — Hammer Fist · Quake Slam · Bull Rush · Berserk
